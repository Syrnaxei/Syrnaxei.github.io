steamctl使用方法

发表于 2026-07-14 分类于 教程

steamctl 项目使用分析

项目简介

steamctl 是由 ValvePython 组织开发的开源命令行工具，功能类似于 Steam 官方的 steamcmd，但它提供了更丰富的 Steam 功能访问能力。它基于 Python 编写，底层依赖 steam 库（同属 ValvePython 生态），可以在终端中完成大量 Steam 相关操作。steamctl主要用于管理和获取 Steam 数据。需要注意的是，中国大陆访问需魔法上网。

安装方式

方式一：通过 PyPI 安装

方式二：直接从 GitHub 安装最新版

安装后直接在终端使用 steamctl 命令即可。

核心功能模块及使用方法

1. apps — 应用信息管理

查询和管理 Steam 应用/游戏信息。

子命令 功能 示例
list 列出账号拥有的游戏或全部应用 steamctl apps list
product_info 查看某款应用的详细信息 steamctl apps product_info <appid>
activate_key 在账号上激活 Steam 密钥 steamctl apps activate_key <key1> <key2>
licenses list 查看账号所有许可证 steamctl apps licenses list
licenses add 添加免费包的许可证 steamctl apps licenses add <packageid>
licenses remove 移除免费包的许可证 steamctl apps licenses remove <packageid>
item_def 获取应用的物品定义 steamctl apps item_def <appid>

2. assistant — 自动化助手

自动完成一些重复性操作。

子命令 功能 示例
idle-games 同时挂机最多 32 款游戏以累积游戏时间 steamctl assistant idle-games <appid1> <appid2>
idle-cards 自动挂机获取游戏卡牌掉落 steamctl assistant idle-cards
discovery-queue 自动浏览一个发现队列（常见于夏促/冬促活动） steamctl assistant discovery-queue

3. authenticator — Steam 验证器管理

这是一个非常实用的功能，可以在命令行中管理 Steam Guard 手机验证器，生成的 token 可以导出到 Steamapp、Aegis 等其他验证器应用。

子命令 功能 示例
add 为 Steam 账号添加验证器 steamctl authenticator add
remove 移除验证器 steamctl authenticator remove
list 列出所有已配置的验证器 steamctl authenticator list
status 查询账号的 Steam Guard 状态 steamctl authenticator status
code 生成当前的验证码 steamctl authenticator code
qrcode 生成验证器的二维码 steamctl authenticator qrcode

4. depot — Depot 下载与浏览

浏览和下载 Steam Depot（Steam 的内容分发存储单元），适合高级用户获取游戏资源文件。

子命令 功能 示例
info 查看 Depot 信息 steamctl depot info <depotid>
list 列出 Depot 中的文件 steamctl depot list <depotid>
download 下载 Depot 文件 steamctl depot download <depotid>
diff 对比清单(manifest)和本地文件的差异 steamctl depot diff <manifest1> <manifest2>
decrypt_gid 解密 manifest gid steamctl depot decrypt_gid <gid>

5. workshop — 创意工坊管理

搜索、浏览、订阅和下载 Steam 创意工坊内容。

子命令 功能 示例
search 搜索创意工坊 steamctl workshop search "skyrim"
info 查看工坊物品详情 steamctl workshop info <workshop_id>
download 下载工坊物品 steamctl workshop download <workshop_id>
subscribe 订阅工坊物品 steamctl workshop subscribe <id1> <id2>
unsubscribe 取消订阅 steamctl workshop unsubscribe <id>
favorite 收藏工坊物品 steamctl workshop favorite <id>
unfavorite 取消收藏 steamctl workshop unfavorite <id>

6. cloud — Steam 云存档管理

管理游戏的云存档、设置等云端文件。

子命令 功能 示例
list 列出某应用的云文件 steamctl cloud list <appid>
list_apps 列出所有有云文件的应用 steamctl cloud list_apps
download 下载应用的云文件 steamctl cloud download <appid>

7. ugc — 用户生成内容(UGC)

获取和下载 UGC 内容详情。

子命令 功能 示例
info 获取 UGC 详情 steamctl ugc info <ugc_id>
download 下载 UGC steamctl ugc download <ugc_id>

8. webapi — Steam WebAPI 访问

直接调用 Steam 的 WebAPI 接口，适合开发者使用。

子命令 功能 示例
set-key 设置 WebAPI 密钥 steamctl webapi set-key <your_key>
clear-key 清除已保存的密钥 steamctl webapi clear-key
list 列出所有可用的 WebAPI 端点 steamctl webapi list
call 调用指定的 WebAPI 接口 steamctl webapi call <interface> <method>

9. hlmaster — 游戏服务器查询

查询 GoldSrc/Source 引擎的游戏主服务器和服务器信息。

子命令 功能 示例
query 查询主服务器获取服务器列表 steamctl hlmaster query
info 查询某台服务器的信息 steamctl hlmaster info <server_addr>

10. steamid — SteamID 解析

解析各种格式的 SteamID。

可以将 SteamID64、SteamID2、SteamID3 等格式互相转换。

11. clear — 清理本地数据

子命令 功能
cache 清除所有缓存和数据文件
credentials 清除所有已保存的登录凭据
all 清除所有缓存文件

典型使用场景

  1. 批量激活 Steam 密钥：steamctl apps activate_key KEY1 KEY2 KEY3
  2. 无手机也能用 Steam 验证器：steamctl authenticator add → 导出 token 到其他验证器 App
  3. 自动挂机刷游戏时间/卡牌：steamctl assistant idle-games 730 440
  4. 下载创意工坊 Mod：steamctl workshop download 12345678
  5. 备份云存档：steamctl cloud download 570（以 Dota2 为例）
  6. 获取游戏 Depot 资源：steamctl depot download <depotid>
  7. 查询服务器信息：steamctl hlmaster query

注意事项

  • 使用大多数功能需要先登录 Steam 账号（首次使用 steamctl 会提示你输入账号密码或通过其他方式认证）
  • authenticator 功能不需要 root 权限，且生成的 token 是可迁移的
  • 依赖 Python 环境，建议使用 Python 3.6+
  • 项目地址：https://github.com/ValvePython/steamctl
